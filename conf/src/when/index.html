Welcome

When is a natural language date/time parser with pluggable rules written in Rust, it's source code can be found here, this server source code can be found here

Try various requests

  • tonight at 11:10 pm
  • at Friday afternoon
  • the deadline is next tuesday 14:00
  • drop me a line next wednesday at 2:25 p.m
  • it could be done at 11 am past tuesday
or pass your own request here handler and see the parse results in JSON format.

Supported get-parameters

input - input string

tz - define a timezone, for instance tz=Europe/Paris (default is Europe/Moscow)

exact_match - enable/disable Damerau-Levenshtein analysis, allows to correct simple typos in input (default is disabled)

dist - maximum merge distance (default is 5)
